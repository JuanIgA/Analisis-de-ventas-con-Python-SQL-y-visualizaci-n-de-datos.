# Análisis de Ventas con SQL y Python

## Descripción del programa:
Este proyecto analiza datos de ventas para identificar ingresos y categorías claves del negocio.

## Herramientas utilizadas:
- Python
- SQLite
- Matplotlib

## Temas analizados:
- Ingresos totales
- Categoría más rentable
- Porcentaje de ingresos por categoría

## Resultados dados:
- Tech representa ~76% de los ingresos
- Hogar representa ~24%
- El negocio depende fuertemente de la categoría Tech

## Conclusión final:
El análisis muestra una fuerte concentración en productos tecnológicos en comparación con la categoría "Hogar". Esto indica que el negocio depende principalmente de la categoría 'Tech', lo que representa una oportunidad para potenciar dicha línea, o un posible riesgo si la demanda decae. Por otro lado, la categoría "Hogar" presenta potencial como una vía alternativa de crecimiento.
