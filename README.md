#Análisis de Ventas con SQL y Python

#Descripción del programa:
Este proyecto analiza datos de ventas para identificar insights clave del negocio.

#Herramientas utilizadas:
- Python
- SQLite
- Matplotlib

#Temas analizados:
- Ingresos totales
- Categoría más rentable
- Porcentaje de ingresos por categoría

#Resultados dados:
- Tech representa ~76% de los ingresos
- Hogar representa ~24%
- El negocio depende fuertemente de la categoría Tech

#Conclusión final:
El análisis muestra una fuerte concentración en productos tecnológicos por sobre la categoría "Hogar",
lo que podría dar a entender que la empresa debería centrarse sobre 'Teach' o, si se quiere, solidificar y buscar puntos fuertes para mejorar
la categoría 'Hogar'.
